# Ecommerce Website (Eshoppy)

A modern, responsive ecommerce clothing web application focused on clean UI, smooth user experience, and scalable frontend architecture.

**Live Demo:** https://eshoppy-website.vercel.app/

---

## UI/UX & Architecture

- Followed **Atomic Design Methodology** (atoms, molecules, organisms)
- Emphasis on **usability, readability, and visual consistency**
- Minimalistic layout with intuitive navigation
